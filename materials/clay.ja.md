# 粘土
粘土を使用することで、精密な造形が可能になります。「直接造形」と「型取り」の手法がありますが、ここでは造形の一般的な特性をまとめます。

| 特性 | 詳細 |
|---|---|
| 成形性 | 細部の表現に使用しやすい。 |
| 重量 | 油粘土は重いが、軽量粘土は発泡ウレタン並みに軽い。 |
| 硬度 | 乾燥後は硬化して形状を保持するが、ひび割れしやすくなる。 |
| 通気性 | 気泡がないため通気性はない。そのため、内部にファンを設置する必要がある。 |
| 耐久性 | 油粘土の耐久性は低いが、軽量粘土は比較的高い耐久性を持つ。 |

## 加工方法
### 芯材（アーマチュア）の構築
粘土の自重による崩壊を防ぐため、アルミ線を使用してフレームを作成します。

### 造形
芯材に粘土を盛り付け、スパチュラや指を使って形を整えます。

### 型取り *樹脂注型の場合
完成した原型をシリコンなどで覆い、雌型を作成します。

### 研磨
硬化後、サンドペーパーやロータリーツールを使用して表面を磨きます。

### マーキング
ケガキ針やスパチュラを使用して位置を印します。これは、左右のパーツの位置を正確に決めるために必要です。
