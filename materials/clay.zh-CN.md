# 黏土

使用黏土可以进行精确的造型。虽然有“直接塑造”和“翻模”等方法，但在此我总结了造型的一般特性。

| 特性 | 详情 |
|---|---|
| 可塑性 | 易于用于表现细节。 |
| 重量 | 油泥较重，但轻质黏土和聚氨酯泡沫一样轻。 |
| 硬度 | 干燥后会变硬并保持形状，但容易产生裂纹。 |
| 透气性 | 没有气泡，因此不透气。必须在内部安装风扇。 |
| 耐久性 | 油泥的耐久性较低，但轻质黏土的耐久性相对较高。 |

## 加工方法
### 骨架制作
为防止因黏土重量导致的塌陷，请使用铝线制作框架。

### 雕塑
将黏土涂在核心材料上，并用刮刀或手指进行塑形。

### 翻模 *以树脂浇铸为例
用硅胶等材料覆盖制作完成的原型，以制作阴模。

### 打磨
硬化后，使用砂纸或旋转工具打磨表面。

### 标记
使用划线针或刮刀标记位置。这对于准确确定左右部件的位置是必要的。
