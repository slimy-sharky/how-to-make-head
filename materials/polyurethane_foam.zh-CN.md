# 聚氨酯泡沫

聚氨酯泡沫是头部制作中最常用的材料。它重量轻，易于加工，且可以通过调整密度来增强其耐用性。

| 特性 | 详情 |
|---|---|
| 密度 | 密度越高，抗下垂能力越强；密度越低，越易于加工。 |
| 硬度 | 聚氨酯泡沫越软，越容易压碎；硬度越高，加工难度越大。 |
| 透气性 | 具有气泡结构，因此透气性极佳。 |
| 吸水性 | 具有气泡结构，因此具有吸水性。 |
| 耐化学性 | 对有机溶剂具有抵抗力，因此可以使用有机溶剂型粘合剂。 |
| 老化 | 会因水解而老化。 |

## 加工方法

### 切割
可以使用大剪刀切割聚氨酯泡沫。

### 粘合
在聚氨酯泡沫的两侧涂抹橡胶类粘合剂，然后将其粘合在一起。

### 削减
使用小剪刀对表面进行精细修剪。

### 标记
使用油性记号笔，可以轻松地在上面绘制线条。
