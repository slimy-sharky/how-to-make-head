# 聚氨酯泡棉 (Polyurethane Foam)

聚氨酯泡棉是枕頭製作中最常用的材料。它重量輕，易於加工，且可以透過調整密度來增強其耐用性。

| 屬性 | 詳細說明 |
|---|---|
| 密度 | 密度越高，抗下陷能力越強；密度越低，則越易於加工。 |
| 硬度 | 聚氨酯泡棉越軟，越容易擠壓變形；硬度越高，加工難度越大。 |
| 透氣性 | 由於具有氣泡結構，因此透氣性良好。 |
| 吸水性 | 由於具有氣泡結構，因此具有吸水性。 |
| 耐化學性 | 對有機溶劑具有抵抗力，因此可以使用有機溶劑型黏著劑。 |
| 老化 | 會因水解作用而劣化。 |

## 加工方法
### 切割
聚氨酯泡棉可以使用大型剪刀進行切割。

### 黏合
在聚氨酯泡棉的兩面塗抹橡膠系黏著劑，然後將其黏合在一起。

### 修剪
使用小型剪刀精細地修剪表面。

### 標記
使用油性簽字筆，即可輕鬆在上面劃線。
